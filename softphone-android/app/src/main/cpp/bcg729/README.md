## bcg729 - G.729A Codec

This directory should contain the bcg729 source files.

Run one of these commands from the project root to download them:

**Windows:**
```
setup_g729.bat
```

**Linux/Mac:**
```
bash setup_g729.sh
```

The bcg729 library is open source (BSD license).
G.729 patents expired in 2017.

Source: https://gitlab.ouvaton.org/ouvaton/bcg729
